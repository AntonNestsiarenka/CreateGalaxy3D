Класс Galaxy описывает галактику в трехмерном пространстве.
При создании экземпляра галактики можно указать следующие параметры:
- имя галактики;
- радиус галактики;
- предел распределения звезд по оси Z (насколько толстый или тонкий диск);
- начало расширения диска галактики;
- количество звезд в галактике;
- координата оси X центра галактики;
- координата оси Y центра галактики;
- координата оси Z центра галактики.

Если данные параметры указаны не будут, то создастся экземпляр галактики по умолчанию с параметрами:
default_RANGE_Z = 1000.0
default_R_GALAXY = 10000.0
default_KOL_STARS = 5000
default_START_EXPANSION = 30
default_DELTA_X = 0
default_DELTA_Y = 0
default_DELTA_Z = 0
При создании экземпляра галактики координаты каждой из звезд генерируются в полярных координатах.
Далее данные координаты приводятся через метод transform_to_a_Cartesian_coordinate_system_XY к декартовой системе кординат. Недостающая координата Z
генерируется отдельно через метод point_sampling_Z, который реализует два вида распределения координаты Z:
1) Часть диска галактики до начала расширения генерируется по формуле z = (triangular(-self.range_z, self.range_z) + self.delta_z).
2) Оставшаяся часть диска рассчитывается через метод up_down_range_z. Данным методом достигается эффект расширения звезд ближе к центру галактики.

Ближе к центру галактики звезды встречаются все реже. В самом центре звезд нет, по задумке там сверхмассивная черная дыра =)
Так же в классе реализован метод rotate_galaxy для вращения всей галактики вокруг заданной оси на заданный угол через матрицу поворота.
Вывод изображения галактики на график выполняется через метод класса build_a_graph. Данный метод способен выводить на один график как одну галактику так и несколько.

Посредтсвом данного класса можно создавать разноообразные виды галактик и системы галактик: по размеру, расположению в пространстве, количеству звезд, началу расширения, толщине диска.
Каждую галактику можно вращать вокруг оси X или Y на заданный угол. Можно создавать различные столкновения двух или более различных галактик.
